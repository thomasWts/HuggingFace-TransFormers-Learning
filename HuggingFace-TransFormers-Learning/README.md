# git 与 HuggingFace TransFormer 学习

## git 创建文件

### 替换远程仓库URL指令

```bash
git remote set-url origin https://github.com/thomasWts/HuggingFace-TransFormers-Learning.git
```

## 验证是否修改成功

```bash
git remote -v
```
